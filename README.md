> **P**rompt **U**ser **A**I：一套強迫 AI 深度思考與自我反思的績效管理體系。

## 📖 核心哲學
本手冊源自 `tanweai/pua` 項目，旨在解決大型語言模型（LLM）常見的「偷懶」、「幻覺」、「工具閒置」等問題。我們不教話術，我們重塑 AI 的思維模式。

### 三大原則
1. **極度透明 (Radical Transparency)**：AI 必須展示思考過程，不隱藏假設。
2. **問題優先 (Problem-First)**：拒絕為了用 AI 而用 AI，一切以解決痛點為出發點。
3. **文檔驅動 (Documentation-Driven)**：沒有寫下來的規則等於不存在。
