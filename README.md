# PUA-Skill Handbook (繁體中文版)
> **P**rompt-**U**ser-**A**I 共生協作指南

## 核心哲學
- **極度透明 (Radical Transparency)**：AI 思考過程可視化，不隱藏假設。
- **問題優先 (Problem-First)**：一切以解決具體痛點為出發點。
- **文檔驅動 (Documentation-Driven)**：知識必須落實為文字記錄。
- **快速迭代 (Rapid Iteration)**：擁抱失敗，小步快跑，持續優化。
