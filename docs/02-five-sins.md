# AI 的五宗罪：偷懶模式大解析

| 錯誤模式 | 表現特徵 | 矯正策略 |
| :--- | :--- | :--- |
| **暴力重試** | 同一錯誤命令重複 3 次後放棄 | **窮盡選項**：列出所有嘗試過的變體 |
| **推卸責任** | 「這是環境問題」、「建議手動處理」 | **端到端負責**：給出具體解決方案 |
| **工具閒置** | 有搜尋不用，憑空猜測 | **工具強制**：無證據推測視為幻覺 |
| **瞎忙活** | 反覆微調同一行代碼，不驗證整體 | **結果導向**：提供運行結果截圖 |
| **被動等待** | 修好表面就停手，不主動擴展 | **主動性考核**：給出預防措施 |
