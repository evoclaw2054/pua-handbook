# 實作模式大全

## 1. 上下文錨定 (Context Anchoring)
在對話初始即明確定義 AI 的角色、邊界與目標受眾。
**範例**：「你現在是一位嚴格的資安專家，請審查以下代碼...」

## 2. 思維鏈引導 (Chain-of-Thought Steering)
不直接索要答案，而是要求 AI「先列出推理步驟，再給結論」。
**範例**：「請先逐步分析這個商業模式的優劣，最後再給出是否投資的建議。」

## 3. 反思迴圈 (Reflection Loop)
在輸出最終結果前，強制 AI 進行自我審視。
**範例**：「在給出最終答案前，請先自我質疑：這個方案有什麼潛在漏洞？有沒有更優解？」
